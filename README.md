#### Problem statement:
 - Create a contact list app to manage contacts.

#### Technology Used:
 - React Js
 - Java Script
 - Redux
 - CSS
 - HTML

 ## Install

To install all the dependences of the project, run the following command:

    git clone https://github.com/simpleesarath/Contact-list-react.git
    npm install
    npm start
